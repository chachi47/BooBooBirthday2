# 🎂 Boo Boo Laand – Birthday Celebrations

A fun, interactive birthday party planner for Boo Boo Laand. Built with React + Vite.

## ✨ Features
- 3-step guided party planner (Welcome → Packages → Add-ons)
- Live price calculator that updates as you pick options
- 4 packages: Whimsical, Legendary, Build Your Own, Lil' Royal's
- Full add-ons menu with categories
- Kids counter with minimum guest enforcement
- Confetti, sounds, and animations 🎉

## 🚀 Deploy to Vercel (Recommended)

1. Push this folder to a GitHub repository
2. Go to [vercel.com](https://vercel.com) and sign in
3. Click **"Add New Project"** → import your GitHub repo
4. Vercel auto-detects Vite — just click **Deploy**
5. Done! You'll get a live URL instantly.

## 🛠 Run Locally

```bash
npm install
npm run dev
```

Then open [http://localhost:5173](http://localhost:5173)

## 📦 Build for Production

```bash
npm run build
```

Output goes to the `dist/` folder — ready to deploy anywhere.
